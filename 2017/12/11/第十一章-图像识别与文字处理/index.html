11.1.2 Tesseract

  • Tesseract 是目前公认最优秀、最精确的开源 OCR 系统

  • Tesseract 也具有很高的灵活性。它可以通过训练识别出任何字体(只要这些字体的风格保持不变就可以,后面我们会介绍),也可以识别出任何 Unicode 字符

  • Tesseract 是一个 Python 的命令行工具,不是通过 import语句导入的库。安装之后,要用 tesseract 命令在 Python 的外面运行

11.2 处理格式规范的文字
